Cyber sec

File Encryption Tool

A secure, client-side utility for AES-256 file encryption and decryption, prioritizing user privacy and data integrity.

Overview

This project provides a robust solution for securing digital assets. Built with the Web Crypto API, it enables high-speed, local-only encryption. No files are ever uploaded to a server—all processing happens directly in the user's browser, ensuring maximum privacy and preventing data leaks.

Key Features

Technical Implementation

Utilizing the Advanced Encryption Standard (AES) in Galois/Counter Mode (GCM) for both confidentiality and authenticity. PBKDF2 is employed for secure key derivation from user-provided passwords.
