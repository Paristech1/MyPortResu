SpeedBump: Outsmarting Philly's Speed Bumps

A specialized navigation solution designed to help drivers navigate Philadelphia's streets more smoothly by avoiding the city's 1,584+ officially installed speed bumps, humps, and cushions.

Use the App Live ↗ View Full Documentation ↗

Key Features

Smart Avoidance Routing

Custom routing engine that prioritizes "Smooth" paths by weighting road segments based on speed bump density and severity.

Official Data Integration

Direct API integration with OpenDataPhilly to fetch real-time updates on traffic calming measures across the city.

Severity Scoring

Accelerometer-based scoring system that allows users to rate the "impact" of specific bumps, crowdsourcing a detailed map of road conditions.
