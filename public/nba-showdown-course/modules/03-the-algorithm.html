The Algorithm

NBA Showdown leaves the final decision up to you, but to help you decide, the app provides a "Game Score" number for everyone.

📊
Composite Metrics

A "composite metric" is a software engineering technique where you merge many variables (Points, Assists, Rebounds) into one single, powerful number. It gives the user an instant, at-a-glance summary instead of making them compare six individual numbers themselves.

The Formula Under the Hood

Look at the specific function in the backend that calculates the Game Score.

CODE 
def _game_score(p):
    return (
        p["PTS"] + p["REB"] * 1.2 + p["AST"] * 1.5
        + p.get("STL", 0) * 2 + p.get("BLK", 0) * 2
        - p.get("TOV", 0) * 1.5
    )
PLAIN ENGLISH

To calculate the game score of player "p"...

Let's tally up the math!

Give 1 point per basket, 1.2 points per rebound, 1.5 per assist.

Give an extra 2 points for every steal or block (defensive bonuses!).

Subtract 1.5 points for every turnover (mistakes matter!).

Add it all up and return the result as the single Game Score number.
