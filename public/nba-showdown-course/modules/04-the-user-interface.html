Module 04

The User Interface

We've fetched the data, calculated the scores, and played the game. But a game only feels premium if it reacts magically to your touch. That's the power of the frontend interface.

✨
Juice

In game design, "juice" is the magical feeling of a button snapping, screen shaking, or card sliding in gracefully. In NBA Showdown, we add juice using a library called Framer Motion.

Buttery Smooth Cards

Look at how the player cards slide into the arena. Instead of instantly teleporting (which feels cheap), they fade in and drift upwards. Let's look at the actual code that orchestrates this dance.

CODE 
<motion.div
  initial={{ opacity: 0, y: 40, scale: 0.95 }}
  animate={{ opacity: 1, y: 0, scale: 1 }}
  exit={{ opacity: 0, y: -40, scale: 0.95 }}
  whileHover={{ scale: 1.02 }}
  whileTap={{ scale: 0.98 }}
  transition={{ duration: 0.28 }}
>
PLAIN ENGLISH

Create a magic box container for our player card...

Before it loads, set it to be invisible, shifted downwards 40 pixels, and slightly shrunk.

When it enters the screen, smoothly fade into reality at its normal size and zero offset.

When it loses the round, neatly drift upwards and fade out completely.

If the user hovers their mouse over it, pop out slightly (1.02x).

When they click, dramatically compress the card inwards (0.98x).

Make all these fluid animations last exactly 0.28 seconds.

End of the magic box declaration.
