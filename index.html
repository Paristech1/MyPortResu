Cyber sec

Kali-Linux Social Engineering Lab: Credential Harvesting Detection

Architected and operated a controlled, cloud-hosted adversary simulation environment on AWS EC2 to replicate real-world credential phishing attacks—enabling hands-on training in attack detection, log forensics, and blue team defensive workflows. The lab leveraged Kali Linux and the Social-Engineer Toolkit (SET) to clone a legitimate web login page and capture simulated credential submissions via HTTP traffic. The infrastructure utilized dynamic Linux user provisioning and XRDP remote desktop configuration to enable GUI-based attack execution and monitoring. The design emphasized enterprise-grade security controls—AWS Security Group hardening, IAM-restricted EC2 access, and ephemeral instance teardown—while maintaining extremely low operating cost during the validation phase.

AWS EC2 Kali Linux Social-Engineer Toolkit (SET) SOC Analysis Incident Response
